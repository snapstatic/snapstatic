  ## What is SnapStatic?

  SnapStatic converts your live Ghost CMS blog into a **fully static website** — pure HTML, CSS, and assets — that loads instantly, scores 100/100 on Lighthouse, and runs on free hosting
  with zero ongoing maintenance.

  No PHP. No Node.js server. No database on the live site. Just fast, secure, static files.

  → **[Generate your site free at snapstatic.io](https://snapstatic.io)**

  ## Features

  ### Free
  - Browser-based generation — no install needed
  - Fetches all posts, pages, tags, authors via Ghost Content API
  - Rewrites internal links and asset URLs for static hosting
  - RSS feed, sitemap.xml, robots.txt included
  - limited generation 500 routes, 3 times in 24h
  - Download as ZIP — deploy anywhere

  ### Pro
  - **Server-side generation** — handles large sites without browser limits
  - **Incremental builds** — only regenerates changed routes
  - **Search** — Fuse.js (zero dependency) or Algolia
  - **Git push deployment** — direct to GitHub, GitLab, or Bitbucket
  - **Custom CDN** — offload assets to your own CDN
  - **HTML minification** — squeeze every byte
  - **File uploads** — bundle static downloads with your site
  - **Archive history** — access every previous build
  - **Build logs** — full visibility into every generation

  → **[See pricing at snapstatic.io/pricing](https://snapstatic.io/pricing)**

  ## How It Works

  Ghost CMS  →  SnapStatic  →  Static ZIP  →  Deploy anywhere
                (fetches via        (HTML + CSS      Cloudflare Pages
                 Content API)        + assets)       GitHub Pages
                                                     Netlify, S3...

  1. Paste your Ghost URL and Content API key
  2. Click **Generate**
  3. Download ZIP or push to Git
  4. Done — your site is live and fast

  **No configuration. No build tools. No deployment pipelines to set up.**

  ## Deployment Targets

  Generated sites are plain HTML/CSS/JS — deploy to any static host:

  - **[Cloudflare Pages](https://pages.cloudflare.com)** — free, global CDN, custom domains
  - **GitHub Pages** — free with your existing repo
  - **Netlify / Vercel** — free tiers, instant deploys
  - **AWS S3 + CloudFront** — enterprise scale
  - **Any VPS** — just serve the folder with nginx
